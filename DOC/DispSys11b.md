# LED displays for System11b

This is an LED display set for later System 11b and 11c machines.

![Disp_Sys11b1](https://github.com/AmokSolderer/APC/blob/master/DOC/PICS/Disp_Sys11b1.jpg)

It contains two 16-digit alphanumerical player displaysand the electronic circuits to drive them.  
I'm using this set with amber LED displays. The color is very close to the original, but they're brighter.

![Disp_Sys11b2](https://github.com/AmokSolderer/APC/blob/master/DOC/PICS/Disp_Sys11b2.jpg)

The Gerber files, BOM and schematics can be found [here](https://github.com/AmokSolderer/APC/tree/master/DOC/Hardware/Sys11b_Display).
